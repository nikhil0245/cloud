What I Learned About Cloud Computing

Cloud computing is a technology that allows users to store data, run applications, and access computing services over the internet instead of relying on local machines. Through this lab, I learned how virtual machines can be created and configured to host websites and services remotely. I also understood the importance of public IP addresses for accessing deployed applications globally. Additionally, cloud platforms improve reliability, scalability, and resource management while reducing hardware dependency and maintenance costs.
